This page records the learning and practice progress following the **First Cloud Journey Bootcamp** roadmap. The content is rewritten according to the worklog template: each week includes main objectives, a detailed daily implementation plan, and an assessment of results.

The roadmap is built based on the FCAJ playlist: program kickoff, drawing AWS architecture, performing workshops, and foundational modules on Cloud, IAM, Budget, VPC, EC2, storage, content distribution, monitoring, and applying knowledge to the final project phase.

**Week 1:** [Kick off Bootcamp, get familiar with Cloud Computing and AWS foundations](1.1-week1/)

**Week 2:** [Set up AWS account, IAM, MFA, and control costs](1.2-week2/)

**Week 3:** [Design VPC network, Subnet, Route Table, and Internet/NAT Gateway](1.3-week3/)

**Week 4:** [Network security, Security Group, NACL, and EC2 connectivity testing](1.4-week4/)

**Week 5:** [Hybrid DNS, Route 53 Resolver, and automation with CloudFormation](1.5-week5/)

**Week 6:** [Connect multiple VPCs using VPC Peering and Transit Gateway](1.6-week6/)

**Week 7:** [Compute on AWS: EC2, AMI, EBS, User Data, and Auto Scaling](1.7-week7/)

**Week 8:** [Backup, Storage Gateway, S3 static website, and CloudFront](1.8-week8/)

**Week 9:** [Synthesize architecture, standardize draw.io diagrams, and prepare project](1.9-week9/)

**Week 10:** [Design cloud application system with serverless approach and centralized data](1.10-week10/)

**Week 11:** [Integrate backend, frontend, monitoring, and business workflow](1.11-week11/)

**Week 12:** [E2E testing, complete report, and clean up resources](1.12-week12/)
